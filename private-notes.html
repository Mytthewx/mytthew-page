Section 1: Introduction do digital transformation with Google Cloud (approximately 10% of the exam) 1.1 Explain why cloud technology is revolutionizing business
  • Define key terms such as cloud, cloud technology, data and digital transformation.
Cloud is a technique, which lets you remotely use servers located in data centers all over the world. Along with it, it gives a possibility to access software that run on those servers. By using Cloud Technology , users do not have to manage physical servers and run software applications locally on their own machine. One of Cloud's benefits is a possibility to access Data from almost any devices, because it is stored in a data center, and we can't so easily lost it. Even a geographic disaster nearby the data center won't be felt by us, if we take care of backups, which is very easy with cloud. Digital Transformation is the adoption of - continously developing - digital technology by an organization. It leds to improved efficiency, value or innovation, all of which are unquestionably beneficial. 1.2 Explain why cloud technology is revolutionizing business
